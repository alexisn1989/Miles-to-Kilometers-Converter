🚗 Miles to Kilometers Converter
A simple and clean Python GUI application that converts
miles to kilometers using Tkinter.
📋 About
This project is a desktop application built with Python's
Tkinter library. Users can enter any distance in miles and
instantly see the equivalent in kilometers with the click
of a button.

✨ Features

Clean and simple GUI interface
Instant conversion with Calculate button
Accurate conversion using 1.609 factor
Lightweight — no external libraries needed

🛠️ Built With

Python 3
Tkinter (built into Python)

🚀 How to Run

Make sure Python is installed
No extra libraries needed!
Clone this repository:

git clone https://github.com/alexisn1989/miles-to-km-converter

Run the program:

python main.py
